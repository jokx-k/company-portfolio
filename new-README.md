# My React Project

A clean, minimal React application built with Vite.

## Getting Started

### Installation
```bash
npm install
```

### Development
```bash
npm run dev
```

### Build for Production
```bash
npm run build
```

## Project Structure
```
src/
  ├── App.jsx          # Main App component
  ├── App.css          # App styles
  ├── main.jsx         # Entry point
  └── index.css        # Global styles
```

Start building your React application!
